# CampusActivityManagementSystem

#### Description
针对日益丰富的校园文化活动，网络报名系统也应该发挥其应有的作用。为此，本小组提出开发一个更加完善的校园活动管理平台，基于微信小程序+springboot + vue 技术构建，用于发布、管理、报名校园活动，核心功能包括活动的发布、管理、报名、参与点名、活动纪念章等。项目包含微信小程序，管理后台。基于java后台语言。
